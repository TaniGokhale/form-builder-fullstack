Project Overview

This is a Dynamic Form Builder System similar to Google Forms, where forms are not hardcoded but created dynamically by an admin. Each form is stored in the database with its own structure and can be accessed via a unique URL.

Users can fill out these forms, and their responses are stored in MongoDB. The admin can view all submissions and analyze data through an analytics dashboard.

 Key Features
Dynamic form creation (text, number, select fields)
Unique shareable form links
Real-time form rendering from backend schema
Secure response submission system
Analytics dashboard with charts and insights
Role-based access (Admin & User)
Clean and scalable backend architecture

 Tech Stack
Frontend: React (TypeScript)
Backend: Node.js, Express.js
Database: MongoDB
State Management: Redux Toolkit / React State
Charts: Recharts / Chart.js



 Concept

The main idea of this project is to handle dynamic schemas, meaning forms are not fixed in the frontend. Instead, the backend sends form structure, and the frontend renders UI automatically based on that structure.
